Product Program Specification
CDLI AI Search and MCP Agent Platform
Project Name: CDLI AI Research Interface
Organization: Cuneiform Digital Library Initiative
Primary Interface: https://cdli.earth
Program Type: AI Research Infrastructure
Core Technology: MCP Agents, LLMs, Semantic Search

1. Executive Summary
The CDLI AI Research Interface introduces an intelligent research layer over the CDLI corpus by integrating LLM-powered agents, Model Context Protocol (MCP) tools, and semantic corpus search capabilities.
The platform will enable scholars to:
Perform natural language queries over the CDLI corpus
Access artifact-level citations automatically
Execute advanced linguistic queries via CQP
Generate research reports and literature summaries

Integrate user-provided LLM APIs
The system will connect existing CDLI infrastructure—including corpus search, advanced search, and the CQP-to-RDF system—through an MCP server that exposes CDLI as a programmable research environment.

2. Program Objectives
Primary Objectives
Introduce AI Search within the CDLI interface.
Provide a standalone AI research workspace for CDLI users.
Enable agentic interaction with the CDLI corpus through MCP.
Integrate CQP-to-RDF linguistic query system into the AI pipeline.

Support automated research writing with artifact citations.
Strategic Goals
Make CDLI the first AI-native digital humanities archive.
Improve accessibility of cuneiform data for researchers and students.
Enable reproducible computational philology workflows.

Foster an ecosystem of AI research tools for ancient text analysis.

The system consists of three primary components.
Component 1 — AI Search Interface
Embedded within the CDLI website.
Features:

Natural language search
Query translation into CDLI parameters
Artifact citation linking
Structured metadata retrieval

Artifact list
translation snippets
links to artifact pages

6. CQP-to-RDF Integration
CDLI includes an experimental corpus query system:
https://cdli.earth/cqp4rdf/
This tool maps Corpus Query Processor (CQP) syntax to RDF graph queries.
The MCP server will expose this capability through an API tool.
Example query:
[lemma="rain"]Response:
Artifact: P254876
Line: Rain invocationThis enables deep linguistic exploration of the corpus.
